## Development

```bash

# install
yarn install

# run
yarn dev
```

visit URL: http://localhost:9527

## Build

```bash
# test environment
yarn build:stage

# production environment
yarn build:prod
```

## Advanced

```bash
# preview
yarn preview

# preview with resource analysis
yarn preview --report

# code format check
yarn lint

# code format check & auto fix
yarn lint --fix
```
